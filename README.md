# Componetes requeridos
1. Descargar e instalar node.js (última version que viene con npm)
2. Descargar e instalar mongoDB y ejecutar el archivo "mongod"
3. Descargar los archivos del repo (o hacer clone)

# Compilar los archivos TypeScript
1. Levantar terminal parado sobre RAIZ/public
2. Ejecutar "npm install"
3. Ejecutar "npm start"

# Levantar el Servidor
1. Levantar terminal parado sobre RAIZ
2. Ejecutar "npm install"
3. Ejecutar "npm start"
